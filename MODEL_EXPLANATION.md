## 🧠 ఏ మోడల్స్ వాడుతున్నాము? (Which Models Are We Using?)

### 1. IndicBERT (ai4bharat/indic-bert)

```
┌─────────────────────────────────────────────────────────────┐
│                     IndicBERT                                │
│                                                              │
│  • 12 Indian languages మద్దతు                               │
│  • తెలుగు, హిందీ, తమిళ్, కన్నడ, మలయాళం...                  │

│  • ALBERT architecture                                       │
│  • Trained on AI4Bharat corpus                              │
└─────────────────────────────────────────────────────────────┘
```

**ఎందుకు IndicBERT?**
- భారతీయ భాషలకు ప్రత్యేకంగా శిక్షణ పొందింది
- తెలుగు భాష బాగా అర్థం చేసుకుంటుంది
- చిన్న సైజు (110M) - మీ M2 Mac లో నడుస్తుంది

## 🔧 మోడల్స్ ఎలా పని చేస్తాయి? (How Models Work)

### అర్కిటెక్చర్ (Architecture)

```
│         "చందమామ రావే జాబిల్లి రావే"                          │

                                  │
                                  ▼
┌─────────────────────────────────────────────────────────────┐

│  ┌─────────────────────────────────────────────────────┐    │
│  │ "చందమామ" → [101, 2315, 4587, 102]                  │    │
│  │ Telugu text → Numbers (tokens)                      │    │
│  └─────────────────────────────────────────────────────┘    │
└─────────────────────────────────┬───────────────────────────┘
                                  │
                                  ▼
┌─────────────────────────────────────────────────────────────┐
│          STEP 2: ENCODER (IndicBERT/MuRIL)                   │
│  ┌─────────────────────────────────────────────────────┐    │
│  │ Tokens → 768-dimensional vectors                    │    │
│  │ ప్రతి పదానికి అర్థం వెక్టర్ గా మారుతుంది             │    │
│  └─────────────────────────────────────────────────────┘    │
└─────────────────────────────────┬───────────────────────────┘
                                  │
                                  ▼
┌─────────────────────────────────────────────────────────────┐
│          STEP 3: OUR CUSTOM MODULES (మా ప్రత్యేక మాడ్యూల్స్)   │
│                                                              │
│  ┌──────────────┐  ┌──────────────┐  ┌──────────────┐       │
│  │    CNN       │  │   Memory     │  │  Feedback    │       │
│  │ (ప్రాస,ఛందస్సు)│  │ (పునరావృతి)   │  │ (మెరుగుపరచు) │       │
│  └──────────────┘  └──────────────┘  └──────────────┘       │
└─────────────────────────────────┬───────────────────────────┘
                                  │

│          STEP 4: DECODER (డీకోడర్)                           │
│  ┌─────────────────────────────────────────────────────┐    │
│  │ Vectors → Telugu text                               │    │
│  │ వెక్టర్స్ → తెలుగు టెక్స్ట్                           │    │
│  └─────────────────────────────────────────────────────┘    │
└─────────────────────────────────┬───────────────────────────┘
                                  │

┌─────────────────────────────────────────────────────────────┐
│                  OUTPUT: సృష్టించిన కవిత                      │
│         "చందమామ రావే జాబిల్లి రావే నీ పాప వచ్చెను..."         │
└─────────────────────────────────────────────────────────────┘
```

## 🎯 మా ప్రత్యేక మాడ్యూల్స్ (Our Novel Modules)

### 1. CNN Module (cnn_module.py)
```python
# ప్రాస మరియు ఛందస్సు నమూనాలను గుర్తిస్తుంది
# Detects praasa and chandassu patterns

Input:  "ఉప్పు కప్పురంబు" 
        "చూడ చూడ రుచులు"
        
CNN →   ప్రాస గుర్తింపు: 'ప్పు', 'డ' (2nd akshara)
        ఛందస్సు: ఆటవెలది (8-10 aksharas)
```

### 2. Memory & Attention Module (memory_attention.py)
```python
# మానవ పునరావృత అభ్యాసం అనుకరిస్తుంది
# Simulates human rote learning

మానవుడు: కవితను 10 సార్లు చదివితే కంఠస్తం అవుతుంది
మోడల్:  పునరావృత నమూనాలను "memory cells" లో నిల్వ చేస్తుంది

Memory cells:
  "చందమామ" → strength: 0.95 (ఎక్కువ పునరావృతి)
  "అద్భుతం" → strength: 0.20 (తక్కువ పునరావృతి)
```

### 3. Feedback Loop (feedback_loop.py)
```python
# ఉత్పత్తిని మెరుగుపరచడం
# Iteratively improves generation

Iteration 1: "చందమామ రావే పాట పాడు"
             ↓ ప్రాస తనిఖీ: FAIL (ప్రాస లేదు)
             
Iteration 2: "చందమామ రావే జాబిల్లి రావే"
             ↓ ప్రాస తనిఖీ: PASS (ప్రాస ఉంది)
             
Output: "చందమామ రావే జాబిల్లి రావే"
```

## 💻 కోడ్ ఉదాహరణ (Code Example)

```python
from src.models.telugu_backbone import create_telugu_generator

# మోడల్ సృష్టించు
model = create_telugu_generator('indicbert')

# కవిత సృష్టించు
prompt = "చందమామ రావే"
generated = model.generate(prompt, max_length=100)

print(generated)
# Output: చందమామ రావే జాబిల్లి రావే నీ పాప వచ్చెను పాలాళి తెచ్చెను
```

## 📈 ఏ మోడల్ ఎంచుకోవాలి? (Which Model to Choose?)

| **Speed** | వేగంగా | నెమ్మదిగా |
| **Memory** | 2GB RAM | 4GB RAM |
| **Quality** | మంచి | అద్భుతం |
